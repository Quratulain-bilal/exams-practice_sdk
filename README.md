# 📚 Exams Practice SDK

A comprehensive SDK for practice exams and test preparation.

## Features

- 🎯 Topic-wise practice questions
- 📊 Performance tracking
- ⏱️ Timed exam simulations
- 📖 Detailed explanations
- 🔄 Spaced repetition support

## Getting Started

```bash
# Install dependencies
npm install exams-practice-sdk

# Or
pip install exams-practice-sdk
```

## Usage

```python
from exams_practice_sdk import PracticeExam

# Initialize exam session
exam = PracticeExam(topic="General")

# Start practice
exam.start()

# Review results
exam.review()
```
